# Test Roadmap — OpenCart Admin Panel Manual UI Testing

## Goal

Ensure stable and reliable admin panel functionality for order
and product management.

---

## Phase 1 — Current coverage
- Order details validation
- Order status update
- Product add, edit, and delete

---

## Phase 2 — Expanded coverage
- Bulk product actions
- Search and filtering edge cases
- Validation of mandatory fields

---

## Phase 3 — Usability and robustness
- Error message clarity
- Form validation behaviour
- Navigation consistency

---

## Phase 4 — Maintenance
- Update test cases when admin UI changes
- Re-execute tests after OpenCart upgrades
- Keep documentation in sync with coverage
